== Description ==

推荐使用 <a href="https://wordpress.org/plugins/mine-cloudvod/">Mine CloudVod 云点播</a>（包含Aliplayer区块），支持Aliplayer组件，配置功能齐全。

任何问题请联系作者咨询，或者在线提交review或者topic，欢迎使用；

== Installation ==
1、把mine-video-aliplayer文件夹上传到/wp-content/plugins/目录下
2、在后台插件列表中激活mine video player - Aliplayer
3、在“Mine视频播放”菜单中输入相关设置信息
